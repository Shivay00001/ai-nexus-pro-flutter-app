# IndividyaBot - AI EdTech Pro

IndividyaBot is a premium, AI-powered EdTech platform built with Flutter and Clean Architecture. It provides a personalized learning experience with an AI Tutor, interactive learning paths, gamification, and advanced AI features like voice validation and knowledge mastery tracing.

## Features

- **AI Tutor**: Interactive chat assistant for personalized learning.
- **Learning Pathways**: Visualized skill trees and personalized paths.
- **Gamification**: Achievements, badges, and progress tracking.
- **Voice Validation**: AI-powered voice answer validation with confidence scoring.
- **Knowledge Mastery**: Detailed analytics and mastery tracing across skills.
- **Offline AI**: Support for on-device inference and model management.
- **Clean Architecture**: Highly maintainable and testable code structure.
- **Riverpod State Management**: Robust and predictable state handling.

## Tech Stack

- **Framework**: Flutter
- **State Management**: Riverpod
- **Navigation**: GoRouter
- **Local Storage**: Hive & Secure Storage
- **Networking**: Dio & Retrofit
- **AI/ML**: Integrated via API and Offline models

## Getting Started

1. Clone the repository.
2. Run `flutter pub get`.
3. Run `flutter pub run build_runner build` to generate models.
4. Run `flutter run`.
